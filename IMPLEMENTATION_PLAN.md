# 한국오션해양테크연구소 웹사이트 통합 개발 계획서

## 1. 프로젝트 개요

- **목표**: 한국오션해양테크연구소의 기업 가치와 두 가지 핵심 브랜드(라캉 왁스, 알마이너)를 효과적으로 전달하는 국/영문 통합 웹사이트 구축.
- **핵심 컨셉**: "해양 기술과 뷰티 사이언스의 융합" + "데이터 기반의 혁신".
- **기술 스택**: Next.js 14, Vanilla CSS (Module), React Context (i18n).

## 2. 상세 기능 명세 및 개발 순서

### 1단계: UI/UX 고도화 및 공통 컴포넌트 구축

> 프리미엄 브랜드 이미지를 위한 디자인 시스템 강화

- **디자인 시스템 정립**: 컬러 팔레트(Beige, Gold, Navy), 타이포그래피(Inter, Playfair Display) 세분화.
- **공통 컴포넌트**:
  - **Button**: Primary, Secondary, Outline, Ghost 스타일 정의.
  - **Card**: 제품 및 뉴스 카드 컴포넌트 (Hover 효과 포함).
  - **SectionLayout**: 페이드인 애니메이션이 적용된 공통 섹션 래퍼.
- **네비게이션 개선**: 스크롤에 따른 헤더 디자인 변화, 모바일 메뉴(햄버거 버튼) 구현.

### 2단계: 라캉 왁스 (Lacan Wax) 페이지 확장

> "프로페셔널을 위한 왁스" 브랜드 아이덴티티 강화

- **제품 쇼케이스**: 기존 쇼핑몰의 제품(하드 왁스, 솔루션 등)을 이미지 갤러리 형태로 소개.
- **특장점 섹션 (USP)**:
  - 저자극 (Hypoallergenic)
  - 강력한 제모력 (Strong Grip)
  - 전문가 전용 (Professional Only)
  - 아이콘과 함께 시각화하여 표현.
- **쇼핑몰 연동**: 각 제품별 "구매하기" 버튼을 통해 기존 쇼핑몰 상세 페이지로 연결.

### 3단계: 알마이너 (Alminer) 페이지 구체화

> 데이터 & 테크 기술력 시각화

- **컨셉 시각화**: "데이터 학습" 및 "AI 분석"을 상징하는 인터랙티브 애니메이션/그래픽 추가.
- **기능 소개 (Features)**:
  - 데이터 마이닝 (Data Mining)
  - 해양 데이터 분석 (Marine Analytics)
  - 스마트 뷰티 솔루션 (Smart Beauty Solution)
- **로드맵 (Roadmap)**: 알마이너의 개발 단계 및 비전을 보여주는 타임라인 UI 구현.

### 4단계: 연구소 소개 (Company) 및 고객 지원

- **연구소 소개**:
  - **비전 & 미션**: 텍스트 + 배경 이미지를 활용한 시차(Parallax) 효과.
  - **오시는 길**: 카카오맵/네이버맵 링크 또는 스타일리시한 지도 이미지 삽입.
- **Footer 보강**: 사업자 정보, SNS 링크, 관련 사이트(Family Site) 드롭다운.

### 5단계: 반응형 최적화 및 SEO

- **모바일 최적화**: 모든 페이지의 모바일 뷰 꼼꼼한 점검 및 스타일 조정.
- **SEO**:
  - 페이지별 `metadata` (Title, Description, Brand Keywords) 설정.
  - Open Graph 태그 설정 (SNS 공유 시 미리보기 이미지).

## 3. 진행 일정 (Workflow)

1.  **공통 컴포넌트 및 레이아웃 개선** (완료)
2.  **라캉 왁스 상세 페이지 구현** (진행 중)
3.  **알마이너 테크 페이지 구현** (완료 - 쿠팡 데이터 기반 고도화)
4.  **반응형 및 최종 폴리싱** (진행 예정)

**비고**:

- 중간 결과물은 언제든 `http://localhost:3100`에서 확인 가능합니다.
